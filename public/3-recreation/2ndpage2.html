Yoga Shorts for Dudes

How these shorts came to be!

I remember finishing my first Yoga class and the feelings that I had. It was like I found something that I knew would be a part of my life for the rest of it. The only problem was that it was hard to find good clothing for the class. The best shorts I would wear to class were board shorts which looked like, well board shorts. So either I walked around the Gym looking like I was going to the beach, or put on athletic shorts which ended up being more revealing than a speedo. So I created a better short which can be worn to the gym, during class, and home after. More lifestyle shorts than Gym shorts. A short which will help you, wherever you are, achieve Zen.
